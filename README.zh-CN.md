## Arbor Coffee — 中英文双语咖啡馆官网

**Arbor Coffee** 是一个中英文双语的精品咖啡店官网，基于 Next.js App Router 与 Tailwind CSS 构建，强调克制、留白与温暖的视觉语言，并提供一致的多页面体验。

### 项目截图

![Arbor Coffee 预览](public/preview.svg)

### 项目亮点

- 中英文双语路由，默认中文首页
- 统一布局与视觉系统（首页 / 菜单 / 故事 / 到店）
- 可访问性：清晰焦点态、减少动画偏好支持
- Tailwind 实现咖啡色系主题与精致排版

### 技术栈

- Next.js（App Router）
- React
- Tailwind CSS
- TypeScript

### 页面路由

- 中文：`/zh`、`/zh/menu`、`/zh/story`、`/zh/visit`
- 英文：`/en`、`/en/menu`、`/en/story`、`/en/visit`

### 本地运行

```bash
npm install
npm run dev
```

浏览器访问 [http://localhost:3000](http://localhost:3000)，默认跳转到中文首页。
