## Arbor Coffee — Bilingual Landing Site

Chinese README: [README.zh-CN.md](README.zh-CN.md)

**Arbor Coffee** is a bilingual (Chinese / English) landing site for a specialty coffee shop. It is built with Next.js App Router and Tailwind CSS, featuring a refined, minimalist visual system and accessible UI.

### Highlights

- Chinese/English routing with default Chinese homepage
- Consistent layout across pages (Home, Menu, Story, Visit)
- Accessible focus states, reduced-motion support
- Tailwind-only styling with a custom coffee color system

### Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- TypeScript

### Routes

- Chinese: `/zh`, `/zh/menu`, `/zh/story`, `/zh/visit`
- English: `/en`, `/en/menu`, `/en/story`, `/en/visit`

### Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). The site will redirect to the Chinese homepage by default.

### Scripts

- `npm run dev` — start development server
- `npm run build` — production build
- `npm run start` — start production server
